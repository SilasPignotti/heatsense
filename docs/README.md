# Documentation

This folder contains short reference notes for the core public components of HeatSense.

## What is documented

- [UrbanHeatIslandAnalyzer.md](UrbanHeatIslandAnalyzer.md) - core UHI analysis engine
- [DWDDataDownloader.md](DWDDataDownloader.md) - DWD weather station download and interpolation
- [CorineDataDownloader.md](CorineDataDownloader.md) - CORINE land cover download
- [WFSDataDownloader.md](WFSDataDownloader.md) - WFS boundary download

## What is intentionally not documented here

- Internal helper functions
- Frontend static assets
- Repetition of configuration already covered in `README.md` and `pyproject.toml`

If you are new to the project, start with the repository `README.md` first.
